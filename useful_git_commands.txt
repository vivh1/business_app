==========================
 GIT COMMANDS QUICK GUIDE
==========================

Clone repository
	git clone https://github.com/vivh1/business_app

Enter repository directory
	cd business_app

Go to main branch
	git checkout main

See all branches (local + remote)
	git branch -a

====================
 FOR EVERY TIME USE
====================

1. Make sure you are on the correct branch
	git checkout branch_name

2. Pull latest changes from remote
	git pull

3. See what changed
	git status

4. Add all modified files
	git add .

5. Commit your changes
	git commit -m "your commit message"

6. Push your changes to remote
	git push

===================
 BRANCH MANAGEMENT
===================

1. Create a new branch
	git checkout -b new_branch_name

2. Switch to an existing branch
	git checkout branch_name

3. Delete a local branch
	git branch -d branch_name

4. Delete a remote branch
	git push origin --delete branch_name

5. View commit history
	git log --oneline --graph --all

6. Fetch updates for all branches
	git fetch --all
